# Church Community Group App

This is a React application designed to help small community groups within a church manage their activities and grow in their faith.

**Features:**

* **Prayer Requests:**
    * Submit and view prayer requests from group members.
    * Mark prayers as answered.
    * Filter prayers by status (answered, unanswered).
* **Upcoming Readings:** 
    * Schedule and track upcoming Bible readings and study materials.
    * Receive reminders for upcoming readings.
* **Meeting Times:**
    * View and manage group meeting schedules.
    * Receive notifications for upcoming meetings.
* **Personal Goals:**
    * Set personal goals (e.g., daily Bible reading, memorizing scripture).
    * Track progress towards goals.
    * Receive encouragement and accountability from group members.
* **User Authentication:**
    * Secure user accounts with Supabase authentication.

**Technologies:**

* **Frontend:** React, JavaScript, Tailwind CSS, Vite
* **Backend:** Supabase, PostgreSQL


**Getting Started:**

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   ```

2. **Install dependencies:**
   ```bash
   cd <project_name>
   npm install
   ```

3. **Create a `.env` file:**
   - Copy the `.env.example` file to `.env`.
   - Replace the placeholder values with your actual Supabase project URL, public and private keys.

4. **Start the development server:**
   ```bash
   npm run dev
   ```
